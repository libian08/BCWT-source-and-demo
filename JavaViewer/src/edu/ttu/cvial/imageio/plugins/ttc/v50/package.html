Provides classes for encoding/decoding images in TTC 5.0 format.

Key new features in TTC 5.0 over 4.1 include:

  • Line-based coding
    Line-based coding has extremely low memory usage and faster coding speed.
  • Region-Of-Interest (ROI) decoding
    A stable ROI decoding without boundary artifacts.
  • ROI-zooming
    A combination of setSourceRegion(), setSourceRenderSize() and setBestFitSourceRenderSize() will decode a ROI best-fitted to the sourceRenderSize.
