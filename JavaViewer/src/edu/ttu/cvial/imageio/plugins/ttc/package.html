Provides abstract classes serving as unified interfaces for all versions of TTC codecs. Each version of TTC codec derives the classes in this package and implements some or all of their methods.

For a specific version of TTC codec, if certain methods are not supported, calls to those methods will be ignored with a warning record in the log.

The code snippet below shows a typical example of reading a TTC image using classes in this package.

// Get an appropriate TTCImageReader.
Iterator imageReaders = ImageIO.getImageReaders(inputStream);
TTCImageReader reader = (TTCImageReader) imageReaders.next();

// Set the input.
reader.setInput(inputStream);

// Get a default set of parameters.
TTCImageReadParam ttcParam = (TTCImageReadParam) reader.getDefaultReadParam();

// Customize the parameters.
ttcParam.setSourceRegion(sourceROI);

// Read the image with customized parameters.
BufferedImage image = reader.read(0, ttcParam);
